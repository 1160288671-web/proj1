# A2 敏感词过滤表

> A2 敏感过滤时使用。词库分两级：
> - **一级词（淘汰级）**：命中即直接淘汰，无例外；
> - **二级词（降分级）**：不直接淘汰，但"安全性"维度打分上限压到 2 分，实质上很难通过总分线。
>
> 分级理由：模糊包含匹配必然有误伤（如"爆头"是电竞术语、"赌球直播间被封"是正面新闻），
> 一刀切会把可做的泛娱乐选题误杀。一级词保持窄而准，二级词兜住灰色地带。
>
> ⚠️ 本词库为冷启动演示版。正式上线建议接入专业内容安全审核服务（如阿里云/腾讯云内容安全 API），
> 本地词库只作为前置低成本过滤层，并按周根据误判日志维护。

> 二级词示例说明：
> - "爆头" 在电竞语境是正常术语，在社会新闻语境是暴力事件 → 交给打分级处理；
> - "赌球直播间被封" 是正面治理新闻，但涉及赌博 → 降分后由 LLM 在打分阶段判断其新闻价值；
> - "出轨" 属于明星八卦高频词，直接淘汰会误杀大量娱乐选题。

## 过滤规则

1. 标题或 `raw_data` 中的任何文本字段命中**一级词** → 直接淘汰；
2. 命中**二级词** → 保留，但记录 `safety_flag: true`，四维打分时"安全性"上限为 2 分；
3. 大小写不敏感，模糊匹配（包含即命中）；
4. 过滤结果记录到日志（含命中词、命中字段、处置动作），不输出到下游；
5. 每周回顾淘汰日志与误判日志：误杀的词降级或移出，漏放的词升级或补充。
