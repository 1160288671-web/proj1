# A6 平台发布配置

> 各平台发布字段规范。A6 仅打包不自动上传。

## 抖音

| 字段 | 限制 | 说明 |
|------|------|------|
| 标题 | ≤ 55 字 | 含话题标签 |
| 简介 | ≤ 1000 字 | |
| 标签 | ≤ 5 个 | #话题格式 |
| 封面 | 1080x1920 JPG/PNG | 含吸引点击的标题文字 |
| 视频 | MP4, H.264, ≤ 4GB | |
