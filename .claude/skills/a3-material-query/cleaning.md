# A3 素材清洗与去重规则

> 全部由代码执行，不走 LLM。

## 1. 字数过滤

- 正文字数 < 500 直接丢弃
- 统计口径：去除 HTML 标签后的纯文本字符数

## 2. 相似度判重

- 使用 embedding 向量计算两两余弦相似度
- 相似度 ≥ 0.85 判为重复簇

## 3. 簇内保留规则（信息密度打分）

满分 9 分，分项加分：

| 特征 | 分值 | 判定方式 |
|------|------|----------|
| 含具体数据/金额/百分比 | +2 | 正则匹配 |
| 含实名人物/机构 | +2 | NER / 正则 |
| 含明确时间线/事件脉络 | +2 | 关键词匹配 |
| 含直接引语 | +1 | 引号标识符 |
| 含因果解释（非纯情绪宣泄） | +2 | 因果关系词匹配 |

## 4. 同分决胜

- 信息密度分相同 → 保留发布时间更早者（原创概率高）

## 5. 输出控制

- 每选题输出 Top 8
- 不足 8 条则在 `materials.json` 中标记 `"material_warning": "insufficient"`
