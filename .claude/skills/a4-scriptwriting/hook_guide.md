# A4 钩子写法指南

> 前 3 秒是完播率的第一道门槛。钩子必须在前 10 个字内抓住注意力。

## 钩子类型

### 1. 信息差型（匹配「优越感」体验）
```
"90% 的人都不知道……"
"你刷到的演唱会说没就没，真正的原因其实是……"
```

### 2. 冲突型（匹配「爽感」体验）
```
"刚官宣就翻车，这次到底谁的锅？"
"他花 200 万买了这个，结果……"
```

### 3. 共鸣型（匹配「共鸣感」体验）
```
"你是不是也发现，最近的演唱会用抢的也买不到票了？"
"所有打工人都听过这句话……"
```

### 4. 问题型（匹配「掌控感」体验）
```
"这个问题不搞清楚，你永远会被割韭菜。"
"一条视频告诉你，为什么最近的演唱会门票突然好买了。"
```

### 5. 反差型（匹配「参与感」体验）
```
"明明是同一个人，三年前和现在判若两人。"
"全网都在骂，但我想说点不一样的。"
```

## 钩子原则

- 不超过 15 个字
- 不用套话（"家人们""咱就是说"）
- 与选题核心冲突直接挂钩
- 棒读风格：口语化、短句、不用复杂从句
