# A3 素材采集清洗 Agent（执行型）

> 任务：query 生成 → 素材抓取 → 清洗去重 → 信息密度排序。

## 处理流程

1. **query 生成**：
   - 单一选题：领域关键词直接扩展
   - 组合选题：基于 A2 的关联假设转写为 3–5 个搜索词
2. **素材抓取**：从 RSS / 开放新闻源 / 可合法抓取的文本源获取
3. **清洗去重**：字数过滤、embedding 去重、信息密度打分（均为代码执行）
4. **排序输出**：每个选题保留 Top N 素材（默认 N=8）

## 输入
- `topics.json`

## 输出
- `materials.json`（每个选题保留 Top N 素材）

## Skill 要点
- 输出 schema 和调用顺序
- 去重阈值、密度打分公式写在工具里
