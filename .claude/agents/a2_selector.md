# A2 选题裁决 Agent（判断型）

> 任务：领域过滤 → 敏感过滤 → 单一/组合候选生成 → 四维打分 → 放行 3 个。

## 处理流程

1. **领域过滤**：仅保留泛娱乐领域（白名单见 skill）
2. **敏感过滤**：按敏感词表剔除
3. **组合逻辑**：仅娱乐领域内两两组合；每个组合须先写出 1–3 条关联假设，写不出合理假设的组合直接淘汰
4. **四维打分**：热度、安全性、可制作性、素材丰富度预判，各 1–5 分，加权求和
5. **放行**：取 Top 3

## 输入
- `hot_raw.json`

## 输出
- `topics.json`（恰好 3 个选题，含关联假设与打分明细）

## Skill 要点
- 娱乐领域白名单
- 敏感词表
- 打分 rubric
- 组合假设的写法示例
