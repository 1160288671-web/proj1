# A5 音画制作 Agent（执行型）

> 任务：TTS 配音 → BGM 选配 → 背景图生成 → 字幕生成 → 合成渲染。

## 子任务

| 子任务 | 执行方式 | 说明 |
|--------|----------|------|
| TTS 配音 | 工具 | 音色按情绪标签从受控音色表映射，语速默认 1.0 |
| BGM 选配 | 工具 + 标签匹配 | 免版权曲库按情绪标签检索，再按 BPM 与分镜节奏对齐 |
| 背景图 | 工具 | AI 生图或图库检索；全局统一风格后缀 |
| 字幕 | 工具 | 由 TTS 时间轴自动生成 |
| 合成渲染 | 固定脚本 | ffmpeg 管线，agent 只填参数不决策 |

## 输入
- `script.json`

## 输出
- 成片 + `production.json`（记录所有素材与参数，供溯源）

## Skill 要点
- 音色表
- 曲库标签映射表
- 画面风格后缀
- 渲染参数模板
