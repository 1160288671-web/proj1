# A4 编剧 Agent（判断型）

> 任务：读素材 → 定体验定位 → 写口播稿 → 拆分镜。

## 核心约束

1. 情绪标签只能从受控词表中选
2. 分镜 schema 字段必须填全
3. 时长预算约束（默认 60–90 秒）
4. 中文 TTS 约 4–5 字/秒，口播稿总字数 = 目标时长 × 4.5（写稿前换算好）

## 体验定位
- 主攻 1 种体验 + 辅攻 1 种体验
- 参照六体验映射表选择
- A2 的 `experience_hint` 可推翻，但必须在 `reason` 字段说明

## 输入
- `topics.json` + `materials.json`

## 输出
- `script.json`（机器交接）+ `脚本报告.md`（人读）

## Skill 要点
- 六体验映射表
- 受控情绪词表
- 分镜 schema
- 钩子写法（前 3 秒）
- 棒读语感要求（口语化、短句）
